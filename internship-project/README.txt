This is the code and poster that I used in my internship at the Singapore Institute of 
Manufacturing Technology (SIMTECH) in 2019 Summer. I used 13 different models to decide which model is 
better to generate a forecast for fast-moving consumer goods.Time series, machine learning forecasts are 
combined using regression-based combination techniques.
